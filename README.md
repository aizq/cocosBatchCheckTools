# cocosBatchCheckTools
合批检查工具

用法：

1、把文件拷贝下来后，挂载在场景任意节点下，

2、点击键盘“Q”，获取当前场景中active为true的（按节点树顺序）所有渲染节点引用的资源路径，可以很直观定位被打断的地方，下图分别是Sprite（console.log）、Label(console.warn)、其他渲染组件(console.error)。
